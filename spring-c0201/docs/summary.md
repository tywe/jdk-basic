#bean自动扫描

##java配置自动扫描bean

@Configuration
@ComponentScan
public class CDPlayerConfig {
}


##xml配置自动扫描bean

###自动包扫描配置：
\<context:component-scan base-package="soundsystem"/\>

###bean注解:
@Component
public class CDPlayer implements MediaPlayer {
    private CompactDisc cd;
    @Autowired
    public CDPlayer(CompactDisc cd) {
        this.cd = cd;
    }
    public void play() {
        cd.play();
    }
}
